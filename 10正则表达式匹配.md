//正则表达式匹配
//使用动态规划
//当p为空，s也为空时匹配成功,否则匹配错误
//根据字符串p最后一位分情况讨论
//当p的最后一位是.时，比较p的倒数第二位与s的倒数第二位是否相等
//当p的最后一位是*时，分情况讨论
//若p最后一位*前的字符出现0次，那就比较p的倒数第三位与s的最后一位是否相等
//若p最后一位*前的字符出现任意次，并且s最后一位与p倒数第二位匹配或者p的倒数第二位是.,那就判断s倒数第二位与p的倒数第二位是否相等
//时间o(mn),空间o(mn)
//动态规划
//由于s是固定会取字符串的，p固定会取字符规律的
//所以我们着重关注字符规律p
//若s为空，p也为空，匹配
//p的最后一个为.或者p与s的最后一个字符相等，判断s与p的倒数第二个字符
//p的最后一个为*时
//假设p的倒数第二个字符出现0次，判断s的最后一个字符与p的倒数倒数第三个字符师傅相等
//假设p的倒数第二个字符出现n次，那就依次判断s的从后向前的字符是否与p的倒数第二个字符相等

/**
正则表达式可以通过？和*匹配相关的字符串，使用正则表达式可以将符合的文件名通过正则biao
*/
class Solution {
    public boolean isMatch(String s, String p) {
        int slen=s.length();
        int plen=p.length();
        boolean[][] res=new boolean[slen+1][plen+1];
        for(int i=0;i<=slen;i++){
            for(int j=0;j<=plen;j++){
    //若s也为空，匹配成功；s不为空，就定义为false;
                if(j==0) res[i][j]=i==0;
                else{
                    if(p.charAt(j-1)!='*'){
                        if(i>=1 && (s.charAt(i-1)==p.charAt(j-1) || p.charAt(j-1)=='.')) res[i][j]=res[i-1][j-1];
                    }else{
//不考虑最后两个字符(就是最后一个*前的字符重复0次，意味着最后两个字符是没用了)
                        if(j>=2) res[i][j] |= res[i][j-2];
//最后一个*前的字符重复N次，那就依次判断s[i-1]……s[0]与p[j-1]是否都匹配；
                        if(i>=1 && j>=2 && (s.charAt(i-1)==p.charAt(j-2) || p.charAt(j-2)=='.')) res[i][j] |= res[i-1][j];
                    }
                }
            }
        }
        return res[slen][plen];
    }
}
